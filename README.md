# WEB2026 - Bohemian Rhapsody (Make It Ugly)

Practice your CSS by creating the most visually offensive design possible.

## Getting started

1. Fork this repository, then clone your fork with GitHub Desktop
2. Open the folder in VS Code
3. Install the Live Server extension (Extensions view, search "Live Server" by Ritwick Dey)
4. Right-click `index.html` and select "Open with Live Server"

## Task

Edit `style.css` to make this website as ugly as possible. Clashing colors, cursed fonts, weird margins, rotated text, cruel hover effects. Go ham.

The one rule: you have to find out how to commit each crime yourself. Search, ask, repeat.

## Files

- `index.html` - The HTML structure (don't modify this)
- `style.css` - Your CSS playground (this is where you work)
- `css-properties.md` - [A menu of possible crimes](css-properties.md)
